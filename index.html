AI Intent Exchange • Universal Playlist File

A playlist file for AI-generated music intent.

Thooster creates a portable AIX Playlist file. It is not a Spotify file, not a YouTube file, and not an Apple Music file. It is a universal music-intent file that future adapters can turn into playlists on any platform.

What this is: a platform-neutral playlist intent file. Future adapters can read this same file and create a playlist in YouTube Music, Spotify, Apple Music, Amazon Music, or another app.
YouTube AdapterFuture connector: search tracks and create YouTube playlist.
Spotify AdapterFuture connector: OAuth login, search tracks, add tracks.
Apple AdapterFuture connector: convert AIX playlist into Apple Music library items.
